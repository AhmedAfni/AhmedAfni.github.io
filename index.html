Featured Work

What I'm building right now.

Flagship Project

DevTrack

DevTrack is a full-stack project management web application built for teams to plan, track, and collaborate on work in a Kanban-style workflow.

React TypeScript Vite TailwindCSS Node.js Express PostgreSQL Prisma ORM
  • Role-based access control with four permission tiers (Owner, Manager, Member, Viewer), backed by JWT authentication
  • Kanban board supporting five distinct task statuses for visualizing workflow progress
  • Team collaboration tools including comments, tags, and an activity feed for tracking changes in real time
  • Notifications system to keep team members updated
  • Sprint tracking for agile-style project planning
  • Analytics dashboard with data visualizations (built using Recharts) to surface project insights
  • Team management via shareable invitation links for onboarding new members
  • Modern, polished UI — a custom GitHub-dark inspired design system for a clean, developer-friendly aesthetic
