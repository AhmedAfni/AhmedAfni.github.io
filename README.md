# Afni Ahamed - Portfolio Website

My personal developer portfolio website hosted on GitHub Pages.

**Live Website:** [ahmedafni.github.io](https://ahmedafni.github.io/)

## 🛠️ Tech Stack
- **HTML** (Structure & content)
- **Tailwind CSS** (via CDN for styling)
- **JavaScript** (For minor interactions/typing effect)

## 📁 Repository Structure
- `index.html` - The main portfolio page containing the layout, sections, and logic.
- `esoft.webp`, `londonmet-3.webp` - Images used in the profile and education sections.

## 🚀 How to Run Locally
Open `index.html` in your browser. Alternatively, use the **Live Server** extension in VS Code.

## 🔄 Deployment (CI/CD)
The site is hosted on GitHub Pages. When code is pushed/merged to the `main` branch, GitHub Actions automatically deploys the updated version.
